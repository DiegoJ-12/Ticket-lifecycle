<h1>osTicket - Ticket Lifecycle: Intake Through Resolution</h1>
The purpose of this project is to demonstrate the full osTicket ticket lifecycle, from intake to resolution, by showing how tickets are created, routed, managed, and closed using proper help desk workflows.<br />



<h2>Environments and Technologies Used</h2>

-Operating System:Windows 10/11
-Server Stack XAMPP(Apache, MySQL, PHP)
-Ticketing System: osTicket
-Database Tool: phpMyAdmin
-Web tools: Chrome Web Browser

<h2>Operating Systems Used </h2>

- Windows 10/11</b>

<h2>Ticket Lifecycle Stages</h2>

- Ticket Creation(Intake)
- Tcket Assignment
- Ticket Acknowlegment
- Troubleshooting & Work Process
- Resolution
- Ticket Closure
- Post-Resolution Review
